Design Guide - Logos List

Edit a CMS page and under it's design tab place the following in the 'layout XML update' field. 
<reference name="content">
     <block type="vaflogo/list" name="vaflogo_list" template="vaflogo/list.phtml" />
</reference>
To display the logos in a "grid" instead of a vertical list, just apply a little CSS: 
.vaflogolist li {
    display: inline-block;
    width: 200px;
}
