Design Guide - Search Results

Grouped Products

If using group view, copy icons to public directory
  • In Elite/Vaf/ copy plus.jpg and minus.jpg to your Magento's index.php path ( usually document root ) - This adds icons on the 'group' mode homepage vaf search

Category Results

In the configuration under the [homepagesearch] section set mode to "category" to get the category view.
In your Magento install path (web root) create a folder called "category-images". In it place a jpg for each category. Example: to display an image for the "Automotive Lights" category create automotive-lights.jpg
