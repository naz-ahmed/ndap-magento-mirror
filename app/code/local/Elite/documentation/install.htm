You must follow "First Steps", then either "Run Installer" or "Manual Installation" (we recommend the installer). Lastly you must follow the instructions under ""Include" JS & CSS".

DISABLE your Magento cache before installing any new modules.
- Paths in Orange refer to the path in your version of Magento, "your files"
- Paths in green refer to the path relative to the extension, "my files".
- Unzip the Elite/ folder to app/code/local/, so you will have app/code/local/Elite.

First Steps (Required)

See Also: Configuration Guide
  • Open app/code/local/Elite/Vaf/
    Rename config.default.ini to config.ini
  • Open app/Mage.php and paste the following ( right at the beginning of the file after the <?php tag ) 
     require_once( 'code/local/Elite/Vaf/bootstrap.php' );
    Note: This needs to be repeated whenever you upgrade Magento. Just refer to the "upgrading magento" link on the table of contents.

Run Installer

Copy Elite/install-vaf.php to Magento's web root. Load it in your browser. Press GO to have the database created and the design files copied. The check boxes allow you to choose which actions to omit if any. The red/green boxes show you if your file permissions are sufficient for installation.

"Include" JS & CSS (Required)

Front End
Paste in some script tags to call in the javascript files from the templates.
Front-end Open up [Magento 1.3.x] app/design/frontend/default/default/template/page/html/head.phtml,
Open up [Magento 1.4.x] app/design/frontend/base/default/template/page/html/head.phtml

copy the following, place _before_ the <title> tag. [1] 
<!-- start VAF -->
<script src="/skin/adminhtml/default/default/jquery-1.4.2.min.js" type="text/javascript" language="javascript"></script>
<script type="text/javascript" language="javascript">
 jQuery.noConflict();
</script>
<script src="/skin/adminhtml/default/default/jquery.metadata.pack.js" type="text/javascript" language="javascript"></script>
<script src="/skin/adminhtml/default/default/jquery.simplemodal-1.3.min.js" type="text/javascript" language="javascript"></script>
<script src="<?=Mage::getBaseUrl( Mage_Core_Model_Store::URL_TYPE_LINK, isset($_SERVER['HTTPS']) )?>vaf/ajax/js?front=1" type="text/javascript" language="javascript"></script>
<link rel="stylesheet" type="text/css" href="/vafStyle.css.php" media="all" />
<!-- end VAF -->
* You may need to edit some of these paths depending on where you installed Magento.
* Due to Magento breaking backwards compatability, you *will* need to edit these paths in "Magento ver. 1.4.2.0"
Back End
Backend Admin Panel. Open up app/design/adminhtml/default/default/template/page/head.phtml, copy & paste the following, place _before_ the <title> tag. 
<!-- start VAF -->
<script src="<?=$this->getSkinUrl()?>jquery-1.4.2.min.js" type="text/javascript" language="javascript"></script>
<script type="text/javascript" language="javascript">
 jQuery.noConflict();
</script>
<script src="<?=Mage::getBaseUrl()?>vaf/ajax/js" type="text/javascript" language="javascript"></script>
<script src="<?=$this->getSkinUrl()?>jquery.metadata.pack.js" type="text/javascript" language="javascript"></script>
<link rel="stylesheet" type="text/css" href="<?=$this->getSkinUrl()?>multiTree.css" />
<!-- end VAF -->

* Due to Magento breaking backwards compatability, you *will* need to edit these paths in "Magento ver. 1.4.2.0"
