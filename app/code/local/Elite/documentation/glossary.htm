Glossary

Vehicle
A given Make/Model/Year.
Vehicle List
The aggregrate of vehicles the system knows about. Also used to refer to a list of vehicles being imported/exported.
Application Fitment
Relationship between product & vehicle definition, also used as a verb to describe the act of creating mappings
Grid View
A search results display style which uses the built in Magento product listing
Group View
An alternative search results display style which uses an expand/collapse javascript interface to group the product results by their category
Category View
An alternative search results display style which uses shows categories that have product matching the selected vehicle, along with category images
Leaf Level
The bottom most or last level, usually "year" if doing make/model/year
Root Level
The top most or first level, usually "make" if doing make/model/year
