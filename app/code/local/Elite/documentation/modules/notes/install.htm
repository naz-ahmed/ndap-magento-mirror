Fitment Notes - Install

Installation

To install the module:

  • First upload the module, upload Elite/Vafnote into your app/code/local/Elite/ path.
  • Then run the SQL bundled with the module. Open Vafnote/sql/Notes.sql and run it in phpmyadmin.
  • You may use the installer, or manually copy Vafnote/Elite_Vafnote.xml to app/etc/modules
