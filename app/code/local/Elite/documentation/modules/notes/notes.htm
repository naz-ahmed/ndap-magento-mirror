Fitment Notes - Notes

Note Codes

Navigate to Vehicle -> Fit Notes -> Manage on the admin menu and set up your notes definitions. Each note gets an 'code' (alphanumeric) and message. You will use these codes later on in your fitment applications import.

Import/Export Note Codes

Navigate to Vehicle -> Fit Notes -> Import (or Export). Use the following template for importing: 

"code","message"
"code1","foo"
"code2","bar"
