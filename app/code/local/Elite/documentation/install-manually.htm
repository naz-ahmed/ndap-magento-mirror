Manual Installation

If the installer script doesn't work (for example, if there are permission issues you were unable to resolve) you can try copying the files by hand. After you copy the files you'll also need to run the sql file located in Elite/Vaf/sql/MMY.sql
  1. Load module XMLs app/code/local/Elite/Vaf/
    Copy Elite_Vaf.xml to app/etc/modules/. Similarly you must do this for each module you purchased.
  2. Install jquery dependencies into your skin path.
    • Copy: Elite/skin/ to: skin/. Overwrite/merge files if/when prompted.
  3. Copy Designs (templates+layouts)
    • Copy Elite/Vaf/design/ to app/design/. Overwrite/Merge Files when prompted.
  4. Copy CSS
    • Copy Elite/Vaf/jsAndCss/vafStyle.css.php to /. (web root).
  5. Copy icons to public directory
    • In Elite/Vaf/ copy plus.jpg and minus.jpg to your Magento's index.php path ( usually document root ) - This adds icons on the 'group' mode homepage vaf search
- Paths in Orange refer to the path in your version of Magento, "your files"
- Paths in green refer to the path relative to the extension, "my files".
- Unzip the Elite/ folder to app/code/local/, so you will have app/code/local/Elite.
